[Objetivo:

Desenvolvimento de um agente inteligente especializado na curadoria e estruturação de dados sobre Sífilis, projetado para dar suporte à escrita de Trabalhos de Conclusão de Curso (TCC) e artigos científicos.

O agente processa fontes bibliográficas fornecidas para gerar insights organizados em tom dissertativo-acadêmico. Mais do que uma simples compilação, a ferramenta prioriza a originalidade e o encadeamento lógico, criando um "fio condutor" que serve como roteiro estruturado para a produção textual, seguindo rigorosamente as normas de escrita de monografias.]

[Objective:
Development of an intelligent agent specialized in the curation and structuring of data on Syphilis, designed to support the writing of undergraduate theses (TCC) and scientific articles. The agent processes provided bibliographic sources to generate insights organized in an academic-dissertative tone. More than a simple compilation, the tool prioritizes originality and logical flow, creating a "guiding thread" that serves as a structured roadmap for textual production, strictly adhering to monograph writing standards.]
